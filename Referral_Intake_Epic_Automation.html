Automation Vendor Solution Bot — Epic Navigation & Referral Logic

Automation Vendor Solution searches for the patient in Epic and applies the correct referral shell logic.

🔀 Automation Vendor Solution Referral Shell Decision Logic — Three Paths
✓ Patient Found + Shell Exists
  • Confirm patient demographics
  • Open existing referral shell
  • Update insurance & coverage
  • Update referring provider info
  • Refresh reason & clinical notes
  • Flag interpreter need if present
  • Upload docs to Media Manager
⊕ Patient Found + No Shell
  • Confirm patient demographics
  • No existing referral shell found
  • Create new incoming referral shell
  • Populate all intake fields
  • Set insurance & coverage
  • Flag interpreter need if present
  • Upload docs to Media Manager
⛔ Insurance Not Accepted
  • Insurance pre-check fails at intake
  • Automation bot does NOT enter Epic
  • Referral flagged in data store
  • Status set to "Insurance Review"
  • Coordinator notified via alert
  • All intake data & docs retained
  • Held until insurance is resolved
